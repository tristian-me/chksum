# A simple checksum program
This is just a quick and simple checksum program which will generate MD5, SHA1, SHA256 and SHA512. This is mostly 
written as a quick way to check from the CLI in Windows.

## Usage
Check directory:
```shell
chksum 
```

Check single file:
```shell
chksum README.md
```

Check multiple files:
```shell
chksum README.md LICENSE
```
